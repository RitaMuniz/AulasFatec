Frequently Asked Questions (FAQ)

What is the purpose of the reforestation system?

The system helps track reforestation data in Brazil over the years. It allows users to add, edit, delete, and view records, as well as generate charts that show how reforestation has evolved.

How can I register a new reforestation value?

To register, enter the desired year and value, then click Save, keep in mind that you can only register years that do not already exist in the system.

Why does a message appear saying the year already exists?

This happens because each year can only have one reforestation record, the system prevents duplicate entries to avoid errors and keep data organized.

Can I edit a reforestation value that has already been registered?

Yes, on the main screen, click the row of the record you want to update, edit the information, and press Edit, the changes are saved automatically.

Can I change the year of an existing record?

No, the year works as a unique identifier for each record, if you need to correct it, delete the incorrect record and register it again with the correct year.

How can I delete a registered value?

Select the corresponding row and click Delete to remove the record from the list.

How can I search for reforestation data?

You can view all records or filter by the last 5 or 10 years, just choose the desired filter, and the system will display the matching results.

Can I track the progress of reforestation?

Yes, the system automatically generates a line chart showing the variation in values over the years. Each point represents the value for a specific year, making it easy to see increases or decreases in reforested areas, the chart updates based on the selected filter (all years, last 5, or last 10).
