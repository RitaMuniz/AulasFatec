Perguntas Frequentes (FAQ)

Qual é o objetivo do sistema de reflorestamento?

O sistema ajuda a acompanhar os valores de reflorestamento ao longo dos anos no Brasil. Com ele é possível cadastrar, editar, excluir e consultar registros, além de gerar gráficos que mostram como o reflorestamento tem evoluído.

Como cadastrar um novo valor de reflorestamento?

Para cadastrar, informe o ano e o valor desejado e clique em Salvar. É importante lembrar que só podem ser cadastrados anos que ainda não tenham registros no sistema.

Por que aparece a mensagem dizendo que o ano já existe?

Isso acontece porque cada ano pode ter apenas um registro de reflorestamento, o sistema bloqueia duplicidades para evitar erros e manter os dados organizados.

Posso editar um valor de reflorestamento já cadastrado?

Sim, na tela principal, clique na linha do dado que deseja alterar, atualize as informações e pressione Editar,as alterações são salvas automaticamente.

Posso alterar o ano de um registro já existente?

Não, o ano funciona como um identificador único do registro. Se for necessário corrigir, exclua o registro incorreto e cadastre novamente com o ano certo.

Como excluir um valor cadastrado?

Selecione a linha correspondente e clique em Excluir para remover o registro da listagem.

Como realizar uma pesquisa pelos dados de reflorestamento?

É possível visualizar todos os registros, apenas os dos últimos 5 anos ou os dos últimos 10 anos, basta escolher o filtro desejado para o sistema exibir os resultados correspondentes.

Posso acompanhar o progresso do reflorestamento?

Sim, O sistema gera automaticamente um gráfico de linhas que mostra a variação dos valores ao longo dos anos. Cada ponto representa o valor de um ano específico, permitindo ver se houve aumento ou redução nas áreas reflorestadas, o gráfico se atualiza conforme o filtro selecionado (todos os anos, últimos 5 ou 10).
