## Aulas Fatec

Esse repositório possui exercícios realizados em sala de aula de diferentes matérias cursadas no Tecnólogo de Análise e Desenvolvimento de Sistemas

Todos os exercícios desse repositório foram realizados entre Jan/2024 - Mar/2026.

Dentre os arquivos, há códigos em: C++, C, HTML, JSP e Java.
O repositório ainda está sendo atualizado com os mais exercícios e com a documentação.
